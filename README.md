## 🔄 Hướng Dẫn Đồng Bộ GitHub Gitless Sync

### GitHub Gitless Sync là gì?

**GitHub Gitless Sync** là plugin Obsidian giúp đồng bộ vault lên GitHub mà **không cần cài Git**. Plugin tự động push/pull mỗi khi bạn chỉnh sửa file.

### Bước 1: Tạo Personal Access Token (PAT)

1. Đăng nhập **GitHub.com** → bấm avatar góc phải → **Settings**
2. Scroll xuống cuối → **Developer settings** (bên trái)
3. Chọn **Personal access tokens** → **Tokens (classic)**
4. Bấm **Generate new token** → **Generate new token (classic)**
5. Đặt tên: `obsidian-sync`
6. Chọn thời hạn: **90 days** hoặc **No expiration** (nếu muốn vĩnh viễn)
7. Tick chọn quyền: ✅ **`repo`** (toàn bộ quyền repo)
8. Bấm **Generate token**
9. **Copy token ngay lập tức** (sẽ không hiển thị lại lần nữa!)

> ⚠️ **Lưu ý:** Token chỉ hiển thị 1 lần. Nếu mất phải tạo lại.

### Bước 2: Cài Plugin trên Obsidian (Máy tính)

1. Mở Obsidian → **Settings** (⚙️) → **Community plugins**
2. Bấm **Browse** → tìm `GitHub Gitless Sync`
3. Bấm **Install** → **Enable**
4. Vào **Settings** → **GitHub Gitless Sync**:
   - **Repository:** `datng3823/tom-tat-luat-dan-su`
   - **Branch:** `main`
   - **Token:** Paste Personal Access Token vừa tạo
   - **Auto pull interval:** `5` (phút)
   - **Auto push on change:** ✅ Bật

### Bước 3: Cài Plugin trên Điện thoại (iOS/Android)

1. Mở Obsidian mobile → **Settings** → **Community plugins**
2. Tìm cài `GitHub Gitless Sync` (tương tự máy tính)
3. Vào **Settings** → **GitHub Gitless Sync**:
   - Điền **cùng token** và **cùng repo** như máy tính
   - **Auto pull interval:** `5` phút
   - **Auto push on change:** ✅ Bật

### Bước 4: Sử dụng hàng ngày

| Hành động | Máy tính | Điện thoại |
|-----------|----------|-----------|
| **Tạo/sửa file** | Tự động push | Tự động push |
| **Muốn lấy bản mới nhất** | Bấm nút Pull (↻) | Bấm nút Pull (↻) |
| **Kiểm tra sync** | Xem icon trạng thái plugin | Xem icon trạng thái plugin |

### Workflow đồng bộ

```
Máy tính (sửa file) → Auto push → GitHub
                                      ↓
                              Auto pull (5 phút)
                                      ↓
Điện thoại (đọc/sửa) ← Auto push ← GitHub
```

### Xử lý lỗi thường gặp

| Lỗi | Nguyên nhân | Cách sửa |
|-----|------------|---------|
| `401 Unauthorized` | Token hết hạn hoặc sai | Tạo lại PAT, paste lại |
| `409 Conflict` | File xung đột | Pull trước khi push |
| `Permission denied` | Token không đủ quyền | Tick lại quyền `repo` |
| `Repository not found` | Sai tên repo | Kiểm tra `datng3823/tom-tat-luat-dan-su` |
